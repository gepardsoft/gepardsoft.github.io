Workbench for web content development

Complete IDE for in-house customer needs.

  • WYSIWYG editor to design an application for their custom hardware.
  • Output the designed application to a valid jQuery mobile based HTML pages.

Metamodel implemented with EMF, editor with GMF, JET templates used for transformation to CSS/JS/HTML.

Integrate customer SDK in Eclipse

The customer wanted their SDK seamlessly integrated with Eclipse. This included custom launch configurations, project natures, build path entries, API filtering and also a custom annotation processor to modify the code parse tree (AST) so that additional, SDK provided artifacts are visible in Eclipse (outline view + editor).

Embedded system analysis tool

ProgressIDE is an environment dedicated to the development of distributed embedded systems using the ProCom component model. We developed the graphical editor for the model, integrated the simulation tool and implemented the visual output for the simulation. http://www.idt.mdh.se/pride/

Flowchart Designer and Debugger

Flowchart Studio integrates flowchart editor, interpreter and debugger. It allows the user to quickly draw a flowchart though a graphical editor, run it and debug it. Based on Eclipse platform for plugin development. http://sourceforge.net/projects/flowchartstudio/

Web Application for invoice approval workflow

A Google App Engine web application for invoice approval workflow. Users within an organization can post invoices, send for approval, approve/reject. Users can draw lines/write comments on the scanned PDF/JPEG invoices via integrated SVG editor.

Bingo game

A bingo gaming website. Users create accounts, generate/buy tickets for scheduled games, join & watch games in real-time. Administrative web interface allows reporting, game scheduling & management.

Custom Data Grid component for the Web

An Adobe Flex based custom data grid capable of visualizing XML structures. Tree-table visualization style, the component supports different sets of columns for the nested levels of data.

Spreadsheet Editor Component for the Web

An Adobe Flex component for creating & editing Microsoft Excel like tables + Java component for document conversion supporting import from XLSX and export to PDF. Rich support for sheets, comments, formulas, frozen rows/columns, etc.

Document Editor component for Web

An Adobe Flex component for creating and editing rich documents in browser + a Java component for document conversion with support for import/export from/to PDF, Docx, XHTML.

Transitional Care for hospitals

A web application to allow post-hospitalization treatment for patients. The application delivers educational content to patients depending on diagnosis and allows doctor to monitor patient data / progress.

MentalNote EMR

MentalNote is a comprehensive electronical medical record (EMR) solution for mental health practices. It provides central storage, distributed access, sharing and management of patient demographic, historical and clinical data. The application provides e-prescribing integration via DrFirst portal. The solution consists of Java backend and Adobe AIR desktop frontent + web interface for administrators.

Custom CRM system

A web based solution for a construction company to manage its customers, contractors, leads and jobs in progress. Calendar of events, automatic mail notifications and various reporting functions. Printing is enabled via export to CSV or plain text.

The application is implemented in a Java using Spring, VAADIN and MySQL DB.
